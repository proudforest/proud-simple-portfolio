# Space | Data Analyst Portfolio

A sleek, minimalist portfolio website for a Data Analyst, featuring a modern design with dark and light mode support.

## Features
- **Responsive Design:** Works across mobile, tablet, and desktop.
- **Theme Toggle:** Switch between Dark and Light modes.
- **Project Showcase:** Grid layout for highlighting key data analysis projects.
- **Tech Stack:** Simple and efficient use of HTML5, CSS3, and Vanilla JavaScript.

## Getting Started
To view the portfolio locally:
1. Clone the repository.
2. Open `docs/index.html` in your web browser.

## Project Structure
- `docs/`: Contains the website source code.
  - `index.html`: Main structure.
  - `style.css`: Styling and theme transitions.
  - `script.js`: Theme toggle functionality.
- `plan saves/`: Documentation of the project planning phase.
